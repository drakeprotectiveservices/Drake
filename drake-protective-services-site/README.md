## Deploying to Vercel

1. Upload this project to a new GitHub repository.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repository.
4. Keep the default settings.
5. Click **Deploy**.
6. In Vercel, go to **Settings → Domains** and add:
   - drakeprotectiveservices.com
   - www.drakeprotectiveservices.com
7. Copy the DNS records Vercel gives you into Cloudflare DNS.
